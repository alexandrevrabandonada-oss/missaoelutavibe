# Memory: features/ciclo-atividade-v0
Updated: 2026-01-29

Ciclo de Atividade v0: Event participation loop with check-in and post-event tracking. Displays EventCycleBanner on /voluntario/hoje when get_my_next_event_prompt returns suggested_stage != 'none' (pre/day_of/post within 36h window). User navigates to /voluntario/agenda/:id?mode=event for EventModePanel with 3 steps: (1) Check-in via checkin_event RPC, (2) Mark contact attendance via mark_event_invite_attended, (3) Post-event form with allowlisted toggles (brought_plus1, qualified_contacts 0-5, new_contacts 0-5) saved via complete_event_participation. Table event_participations stores status (planned/checked_in/completed/skipped) and actions_json. Extended crm_event_invites with attended_at + attendance_marked_by columns. EventParticipationMetricsCard on CoordenadorHoje shows aggregated check-in/completion counts (no PII). Tracking events: event_cycle_banner_shown, event_cycle_banner_clicked, event_checkin_started, event_checkin_completed, event_contacts_attended_marked, event_post_actions_saved, event_completed.
